Claude Code + DeepSeek + Obsidian：从零搭建 AI 知识库

完整教程：部署环境 → 挂载模型 → 安装 Obsidian → 建立知识库体系

第一部分：环境部署

第一步：安装 Node.js

Claude Code 依赖 Node.js 环境，需要先装它。

下载地址： https://nodejs.org/en/download/

安装完成后，打开终端验证：

node --version
npm --version

两条命令都应该返回版本号。

第二步：安装 Claude Code

在终端执行：

sudo npm install -g @anthropic-ai/claude-code

Windows 用户： 不要加 sudo。以管理员身份打开终端，直接执行 npm install -g @anthropic-ai/claude-code。

第三步：验证安装

claude --version

如果显示了版本号（比如 0.x.x），说明 Claude Code 安装成功。

第四步：下载 CC-Switch

CC-Switch 是一个模型网关层，让 Claude Code 可以调用非 Claude 的模型（比如 DeepSeek）。

下载地址： https://ccswitch.io/zh/

第五步：获取 DeepSeek API Key

  1. 打开 DeepSeek 开发者平台：https://platform.deepseek.com/api_keys
  2. 进入 API Keys 页面 → 创建一个新的 Key
  3. 复制保存好（关掉页面就看不到了）

DeepSeek 的 API 价格很低，能力也够用。一般充 10 块钱可以用很久很久。

第六步：把 API Key 关联到 CC-Switch

  1. 启动 CC-Switch（双击或终端运行）
  2. 在配置界面里：添加 DeepSeek → 粘贴 API Key → 选择模型
  3. 保存配置

第七步：测试

确认 CC-Switch 已启动，然后终端运行 claude 启动 Claude Code，随便问一个问题。

如果能正常回复 → ✅ 全部通了

第二部分：安装 Obsidian

第一步：下载并安装 Obsidian

下载地址： https://obsidian.md/download

第二步：打开 Obsidian，创建库

首次打开选择「新建库」，起个名字。打开的界面：左侧文件列表、中间编辑器、右侧面板。

第三部分：建立知识库体系

3.1 笔记的沉淀分区（PARA 体系）

PARA 是 Tiago Forte 提出的知识管理方法，把笔记分成四个区：

分区放什么例子
1. Projects（项目）有明确目标的短期任务视频脚本、教程
2. Areas（领域）长期关注的责任范围健身、滑雪、工作
3. Resources（资源）参考资料电影推荐、书单
4. Archives（归档）已完成的内容已发布的教程

3.2 知识图谱与笔记关联

用 [[笔记名]] 链接到另一篇笔记。当笔记之间有了链接，打开图谱视图，你会看到一张不断延伸的知识网络。

3.3 插件与技能结合

推荐插件：Excalidraw（画图）、Templater（模板）、Tasks（待办）、Dataview（查询）、Obsidian Git（备份）。

第四部分：这套体系的核心优势

  1. 笔记存在你的硬盘里，不会丢 — 纯本地 Markdown，不是别人的服务器
  2. 不需要维护结构 — PARA 帮你分好了
  3. 笔记之间自动产生关联 — 双链 + 图谱，越记越有体系
  4. AI 可以直接操作你的笔记 — 这是 Obsidian 最根本的优势
